# Personal Website

This is a personal website for Zou Haoming. It is designed as a clean online CV and portfolio-style profile with a subtle indie game atmosphere.

## Project Overview

The website presents my personal identity through a calm visual style, simple navigation, and content inspired by independent games, logic puzzles, and thoughtful design. Instead of making the site look like a game website, the design uses quiet colors, card layouts, and clear typography to show my taste and personality.

## Pages

- `index.html` - Home page with a short introduction and personal highlights.
- `about.html` - About page introducing my interests, values, and connection with games.
- `gallery.html` - Gallery page showing design themes and game-related ideas I appreciate.
- `contact.html` - Contact page with QQ, WeChat, and a static contact form layout.

## Features

- Pure HTML and CSS.
- Semantic HTML structure.
- Responsive layout for desktop and mobile screens.
- CSS Grid and Flexbox layout.
- No JavaScript framework.
- No CSS framework.
- Clean, modern, and youth-friendly visual style.

## Design Direction

The design uses a low-saturation color palette, soft borders, card components, and generous spacing. The goal is to create a personal website that feels modern, organized, and quietly connected to indie games and puzzle thinking.
